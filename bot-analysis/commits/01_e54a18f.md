## Features Introduced

### F-001: TimeManager
Abstracts frame/time conversions.

| Method | Purpose |
|--------|---------|
| `seconds_to_pixel_frame()` | Time to video frame |
| `seconds_to_video_latent_index()` | Time to video latent |
| `seconds_to_audio_latent_index()` | Time to audio latent |
| `get_chunk_time_ranges()` | Temporal tile generation |

### F-002: Script Parsing
Parses timestamped scene scripts.

Format: `[MM:SS-MM:SS] DESCRIPTION | audio:SPEC | guide_specs...`

- Time ranges (MM:SS, MM:SS.ms)
- Audio specs: `silent`, `ambient`, `"dialogue"`
- Guide specs: `first:`, `end:`, `middle:`, `MM:SS:`
- Image refs: `$0`, `$1`, filenames

### F-003: MVP Node
`LTXVSceneExtenderMVP` - testable single-chunk generation.
